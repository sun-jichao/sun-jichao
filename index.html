Sublime Text 小插件（添加Hexo用Front-matter信息）

使用 Hexo 作为Blog程序的时候，官方建议使用 hexo new [layout] <title> 的方式创建新文章，之后再编辑创建的md文件。但这样做的前提必须是先打开程序所在目录，再执行创建命令，比较繁琐。
本人的使用习惯是直接在 Sublime Text 中直接打开一个新文件，编写完成后，再复制到 _posts 目录中，所以默认情况下是没有 Front-matter 区域的，感觉还是有些不便，于是就有了下面这个 Sublime Text 小插件。
